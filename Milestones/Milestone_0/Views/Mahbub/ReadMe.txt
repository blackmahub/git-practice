*** READ DATA FROM HTTP FILE SERVER ***

To run the nodejs version of about-me, you need to do the followings:

1. install nodejs from "https://nodejs.org/en/download/"

2. type "cd /git-practice/Milestone_0/Mahbub/nodejs" to go "nodejs" directory

3. type "node http-server.js <port_number>" to run the http file server. for example: node http-server.js 8095

4. in any internet browser, type "http://127.0.0.1:<port_number>/about-me" to see the about-me page. for example: http://127.0.0.1:8095/about-me

5. here the about-me page fetch data using HTTP GET request from HTTP File server.
